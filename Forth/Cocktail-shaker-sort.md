# Cocktail Shaker Sort in Forth

Here's an implementation of the Cocktail Shaker Sort algorithm in Forth:

```forth
\ Cocktail Shaker Sort Implementation in Forth

\ Swap two elements in an array
: swap-elements ( addr1 addr2 -- )
    2@ swap 2! ;

\ Cocktail Shaker Sort algorithm
: cocktail-shaker-sort ( array len -- )
    over 1- >r          \ Save length-1 in return stack
    0 0                 \ left and right pointers
    r>                  \ Get length-1 back
    1+                  \ Add 1 to get the actual length
    0 do                \ Outer loop
        \ Forward pass (left to right)
        2dup < while    \ While left < right
            2dup 1+ 2@ 2@ 2over 2@ < if  \ If array[left] > array[left+1]
                2swap swap-elements  \ Swap elements
            then
            1+              \ Increment left
            2dup < while    \ Continue while left < right
        repeat
        1-              \ Decrement right
        0 0             \ Reset left pointer
        2dup < while    \ While left < right
            2dup 1+ 2@ 2@ 2over 2@ < if  \ If array[left] > array[left+1]
                2swap swap-elements  \ Swap elements
            then
            1+              \ Increment left
            2dup < while    \ Continue while left < right
        repeat
        1-              \ Decrement right
    repeat
    drop drop ;         \ Clean up stack

\ Alternative simpler version using a flag-based approach
: cocktail-sort ( array len -- )
    0 0 0               \ left, right, swapped
    2dup 1- >r          \ Save length-1 in return stack
    r> 0 do             \ Loop until no swaps occur
        0 to swapped    \ Reset swapped flag
        0 to left       \ Reset left pointer
        0 to right      \ Reset right pointer
        2dup 1- to right  \ Set right to length-1
        left right < while
            left 1+ 2@ left 2@ 2over 2@ < if
                2swap swap-elements
                1 to swapped
            then
            1+ left to left
        repeat
        1- right to right
        right left > while
            right 1- 2@ right 2@ 2over 2@ > if
                2swap swap-elements
                1 to swapped
            then
            1- right to right
        repeat
        swapped 0= if leave then  \ If no swaps, we're done
    repeat
    drop drop ;
