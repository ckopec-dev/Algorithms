# Quickhull Algorithm in Forth

Here's an implementation of the Quickhull algorithm in Forth for finding the convex hull of a set of 2D points:

\ Point structure: x y
\ Stack operations for points
: stack-init ( -- )  0 to stack-top ;

\ Stack for points
1000 constant max-points
create point-stack max-points 2* cells allot
: stack-top 0 ;
: stack-push ( x y -- ) 
    point-stack stack-top 2* cells + ! 
    point-stack stack-top 2* 1+ cells + ! 
    1 stack-top +! ;
: stack-pop ( -- x y ) 
    1 stack-top -! 
    point-stack stack-top 2* cells + 1+ @ 
    point-stack stack-top 2* cells + @ ;
: stack-empty? ( -- flag ) 0 stack-top = ;

\ Calculate cross product of three points
\ Returns positive if counter-clockwise, negative if clockwise
: cross-product ( x1 y1 x2 y2 x3 y3 -- cross )
    2swap 2swap  \ ( x3 y3 x1 y1 x2 y2 )
    >r >r          \ ( x3 y3 x1 y1 ) ( x2 y2 )
    r@ * r> * -    \ ( x3 y3 ) ( x1*y2 - x2*y1 )
    r> * r> * -    \ ( x1*y2 - x2*y1 - x3*y2 + x2*y3 )
    ;
