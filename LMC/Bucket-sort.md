# Bucket Sort Algorithm in LMC

Here's an implementation of Bucket Sort in LMC (Little Man Computer) assembly language:

; Sorts an array of numbers using bucket sort approach

    INP         ; Read first number (array size)
    STA SIZE
    LDA ZERO
    STA COUNT   ; Initialize counter
    
    ; Read array elements
READ_LOOP:
    INP         ; Read next number
    STA ARRAY,COUNT
    LDA COUNT
    ADD ONE
    STA COUNT
    LDA COUNT
    LDA SIZE
    SUB COMPARE
    BRP READ_LOOP

    ; Initialize bucket array (assuming max value is 99)
    LDA ZERO
    STA BUCKET_0
    STA BUCKET_1
    STA BUCKET_2
    STA BUCKET_3
    STA BUCKET_4
    STA BUCKET_5
    STA BUCKET_6
    STA BUCKET_7
    STA BUCKET_8
    STA BUCKET_9

    ; Distribute elements into buckets
    LDA ZERO
    STA INDEX
    LDA ZERO
    STA COUNT

DISTRIBUTE_LOOP:
    LDA ARRAY,COUNT
    STA TEMP
    LDA TEMP
    DIVIDE    ; Divide by 10 to get bucket index
    STA BUCKET_INDEX
    LDA BUCKET_INDEX
    ADD BUCKET_0
    STA BUCKET_0
    LDA COUNT
    ADD ONE
    STA COUNT
    LDA COUNT
    LDA SIZE
    SUB COMPARE
    BRP DISTRIBUTE_LOOP

    ; Collect elements from buckets back to array
    LDA ZERO
    STA INDEX
    LDA ZERO
    STA COUNT

COLLECT_LOOP:
    LDA BUCKET_0
    BRZ SKIP_0
    LDA INDEX
    STA ARRAY,COUNT
    LDA COUNT
    ADD ONE
    STA COUNT

    LDA BUCKET_1
    BRZ SKIP_1
    LDA INDEX
    ADD ONE
    STA ARRAY,COUNT
    LDA COUNT
    ADD ONE
    STA COUNT
SKIP_1:
    ; Continue for other buckets...
    LDA ZERO
    STA COUNT
    LDA SIZE
    SUB ONE
    BRZ SORT_DONE

SORT_DONE:
    ; Output sorted array
    LDA ZERO
    STA INDEX

    OUT
    LDA INDEX
    ADD ONE
    STA INDEX
    LDA INDEX
    LDA SIZE
    SUB COMPARE
    BRP OUTPUT_LOOP
