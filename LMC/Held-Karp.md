# Held-Karp Algorithm Implementation in LMC

The Held-Karp algorithm is used to solve the Traveling Salesman Problem (TSP) using dynamic programming with bitmasks. Here's an implementation in LMC (Little Man Computer) assembly language:

```assembly
; Held-Karp Algorithm for TSP in LMC
; Solves TSP using dynamic programming with bitmask representation

    INP         ; Read number of cities (n)
    STA CITYCNT
    LDA #0
    STA DP      ; Initialize DP table
    STA MASK    ; Initialize mask
    STA CURR    ; Current city
    STA BEST    ; Best tour cost
    STA TEMP    ; Temporary storage
    
    ; Initialize distance matrix
    LDA #100    ; Start of distance matrix
    STA DISTPTR
    LDA #0
    STA DISTCNT
    
INITDIST
    LDA DISTCNT
    LDA CITYCNT
    SUB #1
    BRZ DISTDONE
    LDA DISTPTR
    ADD #10
    STA DISTPTR
    LDA DISTCNT
    ADD #1
    STA DISTCNT
    BRA INITDIST
    
DISTDONE
    ; Initialize DP table for all subsets
    LDA CITYCNT
    SUB #1
    STA SUBSETMAX
    
    ; Initialize DP[0][0] = 0 (starting at city 0)
    LDA #0
    STA DP
    LDA #0
    STA DP+1
    
    ; Main Held-Karp algorithm
    LDA #1
    STA SUBSET
    LDA #0
    STA SUBSETCNT
    
SUBSETLOOP
    LDA SUBSETCNT
    LDA SUBSETMAX
    SUB #1
    BRZ SUBSETDONE
    
    ; For each subset, calculate minimum cost
    LDA #0
    STA CITYCNT2
    LDA SUBSET
    STA CURRENTSUBSET
    
CITYLOOP
    LDA CITYCNT2
    LDA CITYCNT
    SUB #1
    BRZ NEXTSUBSET
    
    ; Check if city is in subset
    LDA CURRENTSUBSET
    LDA CITYCNT2
    LDA #1
    LDA CITYCNT2
    LDA #1
