# Gift Wrapping Algorithm in LMC (Little Man Computer)

Here's an implementation of the Gift Wrapping algorithm (also known as Jarvis March) for finding the convex hull of a set of points in LMC:

```assembly
        ORG 100
        INP             ; Input number of points
        STA COUNT
        LDA ZERO
        STA POINTS      ; Initialize points counter
        LDA ZERO
        STA INDEX       ; Initialize index counter
        LDA ZERO
        STA MIN_Y       ; Initialize minimum Y coordinate
        LDA ZERO
        STA MIN_X       ; Initialize minimum X coordinate
        LDA ZERO
        STA TEMP_X      ; Temporary X coordinate
        LDA ZERO
        STA TEMP_Y      ; Temporary Y coordinate

        ; Input all points
INPUT_LOOP:
        LDA POINTS
        STA TEMP1       ; Save points counter
        LDA COUNT
        SUB TEMP1
        BRZ INPUT_DONE  ; If points = count, done
        
        INP             ; Input X coordinate
        STA COORD_X
        INP             ; Input Y coordinate
        STA COORD_Y
