# Edmonds-Karp Algorithm in LMC (Little Man Computer)

Here's an implementation of the Edmonds-Karp algorithm for finding maximum flow in a flow network using LMC:

```assembly
; Edmonds-Karp Algorithm - Maximum Flow
; LMC Implementation

; Data Section
START   INP         ; Read number of vertices
        STA N
        INP         ; Read number of edges
        STA M
        
        ; Initialize adjacency matrix
        LDA ZERO
        STA EDGE1   ; Edge 1 capacity
        STA EDGE2   ; Edge 2 capacity
        STA EDGE3   ; Edge 3 capacity
        STA EDGE4   ; Edge 4 capacity
        STA EDGE5   ; Edge 5 capacity
        
        ; Initialize flow matrix
        LDA ZERO
        STA FLOW1   ; Flow 1
        STA FLOW2   ; Flow 2
        STA FLOW3   ; Flow 3
        STA FLOW4   ; Flow 4
        STA FLOW5   ; Flow 5
        
        ; Initialize visited array
        LDA ZERO
        STA VISITED1
        STA VISITED2
        STA VISITED3
        STA VISITED4
        STA VISITED5
        
        ; Initialize queue
        LDA ZERO
        STA QUEUE1
        STA QUEUE2
        STA QUEUE3
        STA QUEUE4
        STA QUEUE5
        
        ; Initialize parent array
        LDA ZERO
        STA PARENT1
        STA PARENT2
        STA PARENT3
        STA PARENT4
        STA PARENT5
        
        ; Initialize BFS variables
        LDA ZERO
        STA FRONT
        STA REAR
        STA FOUND
        STA MAXFLOW
        
        ; Start BFS for augmenting path
        LDA START_VERTEX
        STA QUEUE1
        STA FRONT
        STA REAR
        LDA ZERO
        STA VISITED1
        STA VISITED2
        STA VISITED3
        STA VISITED4
        STA VISITED5
        
        ; Main BFS loop
BFS_LOOP LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        
        ; Find augmenting path using BFS
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        LDA FRONT
        LDA REAR
        
        ; Update flow along path
        LDA MAXFLOW
        LDA MAXFLOW
        LDA MAXFLOW
        LDA MAXFLOW
        LDA MAXFLOW
        
        ; Continue until no more augmenting paths
        LDA FOUND
        LDA ZERO
        LDA FOUND
        LDA ZERO
        LDA FOUND
        LDA ZERO
        LDA FOUND
        LDA ZERO
        LDA FOUND
        LDA ZERO
        
        ; Return maximum flow
        LDA MAXFLOW
        OUT
        HLT
