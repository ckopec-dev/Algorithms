# Cocke-Younger-Kasami (CYK) Parsing Algorithm in Go

The CYK algorithm is a dynamic programming algorithm for parsing context-free grammars in Chomsky Normal Form (CNF). It determines whether a string can be generated by a given grammar.

## Implementation

```go
package main

import (
    "fmt"
    "strings"
)

// Grammar represents a context-free grammar in Chomsky Normal Form
type Grammar struct {
    // Non-terminals
    NonTerminals map[string]bool
    // Productions: LHS -> RHS (where RHS is either a terminal or two non-terminals)
    Productions map[string][]string
    // Start symbol
    StartSymbol string
}

// CYKTable represents the parsing table
type CYKTable struct {
    table [][]map[string]bool
    n     int
}

// ParseResult represents the parsing result
type ParseResult struct {
    CanParse bool
    Table    *CYKTable
}

// NewGrammar creates a new grammar
func NewGrammar(startSymbol string) *Grammar {
    return &Grammar{
        NonTerminals: make(map[string]bool),
        Productions:  make(map[string][]string),
        StartSymbol:  startSymbol,
    }
}

// AddProduction adds a production rule to the grammar
func (g *Grammar) AddProduction(lhs string, rhs []string) {
    g.NonTerminals[lhs] = true
    g.Productions[lhs] = append(g.Productions[lhs], strings.Join(rhs, " "))
}

// CYKParse performs the CYK parsing algorithm
func CYKParse(grammar *Grammar, input string) *ParseResult {
    n := len(input)
    if n == 0 {
        return &ParseResult{CanParse: false, Table: nil}
    }

    // Initialize the CYK table
    table := make([][]map[string]bool, n)
    for i := range table {
        table[i] = make([]map[string]bool, n)
        for j := range table[i] {
            table[i][j] = make(map[string]bool)
        }
    }

    // Step 1: Fill the diagonal (single character matches)
    for i, char := range input {
        charStr := string(char)
        for lhs, rhsList := range grammar.Productions {
            for _, rhs := range rhsList {
                if rhs == charStr {
                    table[i][i][lhs] = true
                }
            }
        }
    }

    // Step 2: Fill the table using dynamic programming
    for length := 2; length <= n; length++ {
        for i := 0; i <= n-length; i++ {
            j := i + length - 1
            
            for k := i; k < j; k++ {
                // For each possible split point k
                for lhs, rhsList := range grammar.Productions {
                    for _, rhs := range rhsList {
                        // Check if RHS is of the form "A B" (two non-terminals)
                        parts := strings.Fields(rhs)
                        if len(parts) == 2 {
                            first := parts[0]
                            second := parts[1]
                            
                            // Check if both subparts can be derived
                            if table[i][k][first] && table[k+1][j][second] {
                                table[i][j][lhs] = true
                            }
                        }
                    }
                }
            }
        }
    }

    // Step 3: Check if start symbol can derive the entire input
    canParse := table[0][n-1][grammar.StartSymbol]
    
    return &ParseResult{
        CanParse: canParse,
        Table:    &CYKTable{table: table, n: n},
    }
}

// PrintTable prints the CYK parsing table
func (c *CYKTable) PrintTable(input string) {
    fmt.Printf("\nCYK Parsing Table for input: \"%s\"\n", input)
    fmt.Printf("  ")
    for i := 0; i < c.n; i++ {
        fmt.Printf("%3d ", i)
    }
    fmt.Println()
    
    for i := 0; i < c.n; i++ {
        fmt.Printf("%d ", i)
        for j := 0; j < c.n; j++ {
            if j < i {
                fmt.Print("    ")
            } else {
                if len(c.table[i][j]) == 0 {
                    fmt.Print(" .  ")
                } else {
                    var nonTerms []string
                    for nonTerm := range c.table[i][j] {
                        nonTerms = append(nonTerms, nonTerm)
                    }
                    fmt.Printf("[%s] ", strings.Join(nonTerms, ","))
                }
            }
        }
        fmt.Println()
    }
}

// Example usage
func main() {
    // Create a grammar for balanced parentheses
    grammar := NewGrammar("S")
    
    // Add productions for balanced parentheses
    // S -> (S) | SS | ε
    // For CYK, we need to convert to CNF
    // S -> AB | CD | ε
    // A -> (, B -> SB, C -> S, D -> )
    
    // Let's create a simpler grammar for demonstration:
    // S -> AB | BA
    // A -> a
    // B -> b
    grammar.AddProduction("S", []string{"A", "B"})
    grammar.AddProduction("S", []string{"B", "A"})
    grammar.AddProduction("A", []string{"a"})
    grammar.AddProduction("B", []string{"b"})
    
    fmt.Println("Grammar:")
    fmt.Println("S -> AB | BA")
    fmt.Println("A -> a")
    fmt.Println("B -> b")
    fmt.Println()
    
    // Test cases
    testCases := []string{"ab", "ba", "aa", "bb", "aba", "bab"}
    
    for _, input := range testCases {
        result := CYKParse(grammar, input)
        fmt.Printf("Input: \"%s\" -> %s\n", 
            input, 
            map[bool]string{true: "ACCEPTED", false: "REJECTED"}[result.CanParse])
        
        if result.CanParse {
            result.Table.PrintTable(input)
        }
        fmt.Println()
    }
    
    // Example with a more complex grammar
    fmt.Println("=== Complex Grammar Example ===")
    complexGrammar := NewGrammar("S")
    complexGrammar.AddProduction("S", []string{"A", "B"})
    complexGrammar.AddProduction("S", []string{"B", "A"})
    complexGrammar.AddProduction("A", []string{"a"})
    complexGrammar.AddProduction("B", []string{"b"})
    complexGrammar.AddProduction("A", []string{"S", "A"})
    complexGrammar.AddProduction("B", []string{"B", "S"})
    
    fmt.Println("Grammar:")
    fmt.Println("S -> AB | BA | SA | BS")
    fmt.Println("A -> a")
    fmt.Println("B -> b")
    fmt.Println()
    
    result := CYKParse(complexGrammar, "abab")
    fmt.Printf("Input: \"abab\" -> %s\n", 
        map[bool]string{true: "ACCEPTED", false: "REJECTED"}[result.CanParse])
    
    if result.CanParse {
        result.Table.PrintTable("abab")
    }
}
```

## How the Algorithm Works

### 1. **Initialization**
- Create an n×n table where n is the length of the input string
- Fill the diagonal: for each character, find non-terminals that can generate it

### 2. **Dynamic Programming**
- For each substring length from 2 to n:
  - For each possible starting position:
    - Try all possible split points
    - For each production rule, check if the left-hand side can be derived from the two parts

### 3. **Final Check**
- The input is accepted if the start symbol can derive the entire string

## Key Features

- **Time Complexity**: O(n³) where n is the length of input
- **Space Complexity**: O(n²) for the parsing table
- **Input Format**: Works with strings over any alphabet
- **Grammar Requirement**: Must be in Chomsky Normal Form
- **Output**: Determines whether the string is accepted by the grammar

This implementation demonstrates the core CYK algorithm for parsing context-free grammars, which is particularly useful for parsing languages with nested structures like balanced parentheses or arithmetic expressions.
